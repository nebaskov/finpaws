## Задача

Управление личными финансами требует дисциплины, регулярного анализа и планирования. Большинство людей либо не ведут бюджет вовсе, либо бросают через пару недель из-за рутинности процесса. Существующие приложения предлагают ручной ввод и статичные отчёты, но не дают персонализированных рекомендаций и не адаптируются под привычки пользователя.

FinPaws - агентная система, которая помогает вести бюджет в диалоговом формате: категоризирует расходы, строит планы на неделю/месяц/год, предупреждает о рисках перерасхода и предлагает стратегии накопления. Общается в стилистике кота-финансиста.

## Что делает PoC

- Приём и категоризация транзакций через диалог
- Краткосрочное планирование: недельный/месячный бюджет с лимитами по категориям
- Долгосрочное планирование: цели накопления, прогноз достижения
- Анализ паттернов трат и персонализированные рекомендации
- Мультивалютность с автоматической конвертацией через API курсов
- Сохранение истории и контекста пользователя между сессиями
- Регистрация, авторизация и многопользовательский режим
- Импорт таблиц с тратами и доходами (CSV/Excel) за произвольный период

## Out of scope

- Интеграция с банковскими API и автоматический импорт транзакций
- Инвестиционные рекомендации и работа с ценными бумагами
- Мобильное приложение, веб-интерфейс (только CLI/API)
- Налоговое планирование
